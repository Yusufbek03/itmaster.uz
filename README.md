# IT Master - Профессиональный ремонт компьютеров

Сайт-визитка для сервисного центра по ремонту компьютеров и ноутбуков в Намангане.

## Особенности проекта
- **Мультиязычность**: Поддержка Русского и Узбекского языков (RU/UZ).
- **Telegram Уведомления**: Интеграция с Telegram Bot API для мгновенного получения заявок.
- **Современный дизайн**: Адаптивная верстка, анимации в стиле Apple, стеклянный интерфейс (Glassmorphism).
- **Интерактив**: Модальные окна с детальным описанием услуг, плавная прокрутка, анимация счетчиков.

## Технологии
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- Font Awesome (Иконки)
- Telegram Bot API

## Как настроить Telegram уведомления
1. Получите токен у [@BotFather](https://t.me/BotFather).
2. Узнайте свой Chat ID через [@userinfobot](https://t.me/userinfobot).
3. Вставьте эти данные в начало файла `script.js`:
   ```javascript
   const TELEGRAM_TOKEN = 'ВАШ_ТОКЕН';
   const TELEGRAM_CHAT_ID = 'ВАШ_ID';
   ```
